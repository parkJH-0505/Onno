## 1. 핵심 User Journey (개인화 포함)

### 1-1. 전체 사용자 여정 맵 (개인화 레이어 추가)

```
┌─────────────────────────────────────────────────────────────┐
│  Phase 1: 온보딩 (First-time User) + 개인화 시작             │
└─────────────────────────────────────────────────────────────┘
회원가입 → 프로필 설정 → 관심 도메인 선택 → 첫 회의 튜토리얼
         → "나만의 온노" 소개 → Lv.1 시작
         → Notion 연동 (선택) → 샘플 회의 체험

👤 사용자 상태: Lv.1 (모든 도메인)
💡 AI 행동: 기본 질문 제안 (모든 사용자 동일)

┌─────────────────────────────────────────────────────────────┐
│  Phase 2: 회의 전 (Pre-meeting) + 맥락 로드                  │
└─────────────────────────────────────────────────────────────┘
대시보드 접속 → 레벨 현황 확인 (다음 레벨까지 N회의)
             → 새 회의 시작 → 회의 정보 입력
             → 참석자 선택 (과거 대화 자동 로드)
             → 체크리스트 선택/생성
             → 회의 시작

👤 사용자 상태: 투자 심사 Lv.3 (과거 30회 회의)
💡 AI 행동: 사용자 스타일 반영 (과거 선호 질문 우선)

┌─────────────────────────────────────────────────────────────┐
│  Phase 3: 회의 중 (During-meeting) ⭐ 핵심 경험 + 학습       │
└─────────────────────────────────────────────────────────────┘
실시간 전사 → AI 질문 제안 (개인화됨) → 사용자 선택 (사용/무시)
          → 🎓 AI 학습 (사용 패턴 기록)
          → 인사이트 카드 표시 (레벨에 따라 다름)
          → 액션 아이템 자동 감지 → 승인
          → 회의 종료

👤 사용자 행동: 질문 5개 중 3개 사용, 2개 무시
💡 AI 학습: 사용한 질문 타입 선호도 ↑, 무시한 타입 선호도 ↓

┌─────────────────────────────────────────────────────────────┐
│  Phase 4: 회의 후 (Post-meeting) + 레벨업                    │
└─────────────────────────────────────────────────────────────┘
회의 요약 자동 생성 → 리포트 확인 → 놓친 질문 검토
                  → 📈 경험치 획득 (+20 XP)
                  → 🎉 레벨업! (Lv.3 → Lv.4)
                  → 새 기능 언락 안내
                  → Notion 자동 업데이트 확인
                  → 다음 회의 예약 (선택)

👤 사용자 상태: 투자 심사 Lv.4 달성!
💡 AI 변화: 예측 질문, 패턴 인사이트 기능 언락

┌─────────────────────────────────────────────────────────────┐
│  Phase 5: 일상 사용 (Daily Usage) + 성장 확인               │
└─────────────────────────────────────────────────────────────┘
과거 회의 검색 → 인사이트 분석 → 통계 확인
              → 📊 레벨/도메인 성장 확인
              → 🎭 페르소나 변경 (Phase 3부터)
              → 설정 조정

👤 사용자 상태: 3개월 사용, 50+ 회의, 다중 도메인 Lv.4+
💡 AI 상태: 사용자를 "이해"하는 파트너
```

### 2-2. 샘플 회의 체험 (인터랙티브 튜토리얼)

```
튜토리얼 플로우:

1. 미리 녹음된 데모 회의 재생
   "A팀: 저희 CAC는 $30입니다..."

2. AI 질문 제안 애니메이션
   🔴 "LTV는 얼마인가요?" 카드 등장
   → "이런 식으로 실시간 질문을 제안합니다"

3. 사용자 인터랙션 유도
   → [사용하기] 버튼 클릭 유도
   → "좋아요! 이렇게 사용하면 Onno가 학습합니다"

4. 경험치 획득 애니메이션
   → "+20 XP" 애니메이션
   → "첫 회의 완료! 곧 Lv.2가 될 거예요"

5. 온보딩 완료
   → "이제 진짜 회의를 시작해보세요"
   → [대시보드로 이동]
```
