# Test Data

이 폴더는 프로토타입 테스트를 위한 오디오 샘플들을 저장합니다.

## audio-samples/

테스트용 한국어 비즈니스 대화 녹음 파일을 저장하는 폴더입니다.

### 필요한 샘플:
1. `sample-1-vc-pitch.mp3` - VC 투자 심사 대화 시뮬레이션 (1-3분)
2. `sample-2-mentor-call.mp3` - AC 멘토링 대화 (1-3분)
3. `sample-3-sales-call.mp3` - CB 세일즈 콜 (1-3분)

### 오디오 요구사항:
- 형식: MP3, WAV, 또는 WebM
- 길이: 1-3분
- 언어: 한국어
- 내용: 비즈니스/스타트업 관련 대화

### 주의사항:
- 실제 오디오 파일은 .gitignore에 포함되어 있어 Git에 커밋되지 않습니다
- 개인정보가 포함된 실제 회의 녹음은 사용하지 마세요
- 테스트용으로만 사용하세요
